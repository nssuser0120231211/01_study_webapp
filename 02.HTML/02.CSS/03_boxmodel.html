📦 CSSボックスモデル

まず、CSSのボックスモデルを理解しましょう。すべての要素は4つの領域から構成されています。

Margin（外側の余白）
Border（境界線）
Padding（内側の余白）
Content（内容）
/* ボックスモデルの構成 */ .element { margin: 20px; /* 外側の余白 */ border: 2px solid #333; /* 境界線 */ padding: 15px; /* 内側の余白 */ width: 200px; /* コンテンツの幅 */ height: 100px; /* コンテンツの高さ */ }

Marginは要素の外側に空間を作ります。他の要素との距離を調整するために使用します。

個別に設定
/* Marginの様々な指定方法 */ .margin-example { margin: 20px; /* 全方向 */ margin: 10px 20px; /* 上下 左右 */ margin: 10px 15px 20px 25px; /* 上 右 下 左 */ /* 個別指定 */ margin-top: 10px; margin-right: 15px; margin-bottom: 20px; margin-left: 25px; /* 中央揃え */ margin: 0 auto; /* 水平中央揃え */ }

🟢 Padding（パディング）- 内側の余白

Paddingは要素の内側に空間を作ります。コンテンツと境界線の間の距離を調整します。

📊 Paddingの基本例

Paddingなし：

狭い内容

Paddingあり（padding: 30px）：

ゆとりのある内容

📝 Paddingの書き方

一括指定

padding: 15px;
全方向15px

上下・左右

padding: 20px 30px;
上下20px、左右30px

個別指定

padding: 10px 20px 30px 40px;
上右下左の順

方向別指定

padding-left: 50px;
左側のみ50px
/* Paddingの様々な指定方法 */ .padding-example { padding: 15px; /* 全方向 */ padding: 20px 30px; /* 上下 左右 */ padding: 10px 20px 30px 40px; /* 上 右 下 左 */ /* 個別指定 */ padding-top: 10px; padding-right: 20px; padding-bottom: 30px; padding-left: 40px; }

⚠️ よくある間違いと注意点

1. Marginの相殺（Margin Collapsing）

垂直方向の隣接するmarginは相殺されることがあります。

上の要素（margin-bottom: 30px）
下の要素（margin-top: 20px）
※実際の間隔は30pxになります（大きい方が適用される）
/* Marginの相殺例 */ .element1 { margin-bottom: 30px; } .element2 { margin-top: 20px; } /* 実際の間隔は30px（50pxではない） */

2. Gapは全てのレイアウトで使えない

Gapはdisplay: flexまたはdisplay: gridでのみ動作します。

/* ❌ 間違い：通常のブロック要素でgapは効かない */ .normal-div { gap: 20px; /* 効果なし */ } /* ✅ 正しい：FlexboxまたはGridで使用 */ .flex-container { display: flex; gap: 20px; /* 効果あり */ } .grid-container { display: grid; gap: 20px; /* 効果あり */ }

📐 使える単位の種類

px（ピクセル）

固定サイズ。画面サイズに関係なく常に同じ大きさ。

margin: 20px;

%（パーセント）

親要素のサイズに対する割合。レスポンシブデザインに最適。

padding: 5%;

em

現在の要素のフォントサイズを基準とした相対単位。

margin: 1.5em;

rem

ルート要素（html）のフォントサイズを基準とした相対単位。

padding: 2rem;

vw/vh

ビューポート（画面）の幅/高さに対する割合。

margin: 5vw;

auto

ブラウザが自動計算。centeringなどに使用。

margin: 0 auto;
