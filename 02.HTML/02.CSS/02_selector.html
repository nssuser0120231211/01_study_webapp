🎯 基本セレクタ 基本

1. 要素セレクタ (Element Selector)

HTML要素名を直接指定してスタイルを適用します。

/* すべてのh1要素に適用 */ h1 { color: blue; font-size: 24px; } /* すべてのp要素に適用 */ p { margin: 10px 0; }

見出し1（h1要素）

段落テキスト（p要素）

もう一つの段落テキスト

2. クラスセレクタ (Class Selector)

class属性を持つ要素にスタイルを適用します。「.」で始まります。

/* クラス"highlight"を持つ要素 */ .highlight { background-color: yellow; font-weight: bold; } /* クラス"error"を持つ要素 */ .error { color: red; }
<p class="highlight">ハイライトされたテキスト</p> <p class="error">エラーメッセージ</p> <p>普通のテキスト</p>

ハイライトされたテキスト

エラーメッセージ

普通のテキスト

3. IDセレクタ (ID Selector)

id属性を持つ要素にスタイルを適用します。「#」で始まります。

/* ID"header"を持つ要素 */ #header { background-color: #4CAF50; padding: 20px; color: white; } #footer { text-align: center; font-size: 12px; }
ヘッダー（id="header"）

🔗 組み合わせセレクタ 中級

1. 子孫セレクタ (Descendant Selector)

スペース（ ）で区切って、ある要素の中にある要素を指定します。

/* divの中のpに適用 */ div p { color: green; } /* .containerの中のh2に適用 */ .container h2 { border-bottom: 2px solid blue; }

コンテナ内の見出し

div内の段落テキスト

div外の普通の段落

2. 子セレクタ (Child Selector)

「>」で区切って、直接の子要素のみを指定します。

/* divの直接の子のpのみに適用 */ div > p { font-weight: bold; color: red; }

直接の子要素（太字）

孫要素（太字にならない）

3. 隣接兄弟セレクタ (Adjacent Sibling)

「+」で区切って、直後の兄弟要素を指定します。

/* h2の直後のpに適用 */ h2 + p { font-size: 18px; color: red; }

✨ 疑似クラスセレクタ 中級

1. 動的疑似クラス

/* マウスホバー時 */ button:hover { background-color: #667eea; color: white; cursor: pointer; } /* フォーカス時 */ input:focus { background-color: #007bff; } /* クリック時 */ a:active { color: red; }


リンク01

2. 構造的疑似クラス

/* 最初の子要素 */ li:first-child { color: red; } /* 偶数番目の要素 */ li:nth-child(2n) { background-color: #f0f0f0; } /* 最後の子要素 */ li:last-child { font-weight: bold; }
  • 1番目のアイテム（赤字・背景色）
  • 2番目のアイテム（背景色）
  • 3番目のアイテム
  • 4番目のアイテム（最後・太字・背景色）

🔧 属性セレクタ 上級

/* type属性を持つ要素 */ [type] { border: 2px solid green; } /* type="text"の要素 */ [type="text"] { background-color: #e6ffe6; } /* class属性に"btn"を含む要素 */ [class*="btn"] { padding: 8px 16px; } /* href属性が"https"で始まる要素 */ [href^="https"] { color: green; }

⚖️ セレクタの詳細度（優先度）

複数のセレクタが同じ要素に適用される場合、詳細度の高いものが優先されます。
