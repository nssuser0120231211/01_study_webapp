CSSの基本

CSSの基本を実際の例で学んでみよう！

🎨 CSSとは何か？

CSS（Cascading Style Sheets）は、ウェブページの見た目やデザインを決めるための言語です。

普通のテキストです。

📝 CSSの基本的な書き方

CSSは「セレクタ」「プロパティ」「値」の3つの要素で構成されます：

セレクタ1 {
  プロパティ1: 値1;
  プロパティ2: 値2;
  プロパティ3: 値3;
}

セレクタ2 {
 プロパティ1: 値1;
 プロパティ2: 値2;
 プロパティ3: 値3;
}

/* 実際の例 */
h1 {
  color: blue;
  font-size: 24px;
}

実際の例：

普通のテキスト

赤いテキスト (color: red;)

大きくて太いテキスト (font-size: 24px; font-weight: bold;)

背景が青いテキスト (background-color: lightblue;)

🛠 よく使うプロパティ

文字に関するプロパティ：

color: red; （文字の色）

font-size: 20px; （文字の大きさ）

font-weight: bold; （文字の太さ）

text-align: center; （文字の配置）

背景に関するプロパティ：

background-color: yellow;

background-color: #ffb3ba; （色コード）

レイアウトに関するプロパティ：

width: 200px; height: 50px;
margin: 10px; padding: 10px;
border: 2px solid green;

📚 学習のコツ

1. 小さく始める

まずは色や文字サイズの変更から始めましょう。

2. 実際に書いてみる

読むだけでなく、実際にコードを書いて結果を確認することが大切です。

3. 開発者ツールを使う

ブラウザでF12キーを押すと、リアルタイムでCSSを編集・確認できます。

4. 練習あるのみ

最初は難しく感じるかもしれませんが、練習すればきっと上達します！
