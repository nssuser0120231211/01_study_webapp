📚 オブジェクトって何？

🏠 オブジェクト = 情報の入れ物

オブジェクトは関連する情報をひとまとめにして管理するための「箱」のようなものです。
例えば、「人」という情報を表すときに、名前、年齢、職業などを一つにまとめることができます。

👤 person オブジェクト
name: "太郎"
age: 25
job: "エンジニア"
greet(): 挨拶する
🚗 car オブジェクト
brand: "Toyota"
color: "red"
speed: 0
start(): エンジン始動

🔧 オブジェクトの作り方

基本的な作り方

オブジェクトリテラル（一番簡単！）
// 波括弧 {} を使ってオブジェクトを作る const person = { name: "太郎", // プロパティ（データ） age: 25, city: "東京", greet: function() { // メソッド（動作） return "こんにちは、" + this.name + "です！"; } }; console.log(person.name); // "太郎" console.log(person.greet()); // "こんにちは、太郎です！"

プロパティにアクセスする方法

2つの方法があります
// 1. ドット記法（よく使う） console.log(person.name); // "太郎" console.log(person.age); // 25 // 2. ブラケット記法（特殊な場合に使う） console.log(person["name"]); // "太郎" console.log(person["age"]); // 25 // 変数を使ってアクセス const key = "name"; console.log(person[key]); // "太郎"

プロパティの追加・変更・削除

オブジェクトは後から変更できる
const car = { brand: "Toyota", color: "red" }; // 新しいプロパティを追加 car.year = 2023; car.isRunning = false; // 既存のプロパティを変更 car.color = "blue"; // プロパティを削除 delete car.year; console.log(car); // {brand: "Toyota", color: "blue", isRunning: false}

🎯 「this」って何？

thisは「今のオブジェクト」を指す特別なキーワード

メソッド（オブジェクトの中の関数）の中で、「このオブジェクト」を参照するために使います。

thisの使い方
const student = { name: "花子", grade: 85, subject: "数学", // thisを使ってオブジェクト内の他のプロパティにアクセス introduce: function() { return "私は" + this.name + "です。" + this.subject + "の成績は" + this.grade + "点です。"; }, study: function() { this.grade += 5; // 成績を上げる return this.name + "さんが勉強して、成績が" + this.grade + "点になりました！"; } }; console.log(student.introduce()); // "私は花子です。数学の成績は85点です。" console.log(student.study()); // "花子さんが勉強して、成績が90点になりました！"

🌟 実用的な使い方

ショッピングカートのオブジェクト
const shoppingCart = { items: [], total: 0, // 商品を追加 addItem: function(name, price) { this.items.push({name: name, price: price}); this.total += price; return name + "をカートに追加しました（￥" + price + "）"; }, // 合計金額を表示 getTotal: function() { return "合計金額: ￥" + this.total; }, // カートの中身を表示 showItems: function() { if (this.items.length === 0) { return "カートは空です"; } let result = "カートの中身:\n"; for (let i = 0; i < this.items.length; i++) { result += "- " + this.items[i].name + " (￥" + this.items[i].price + ")\n"; } return result + this.getTotal(); } }; // 使用例 console.log(shoppingCart.addItem("りんご", 150)); console.log(shoppingCart.addItem("パン", 200)); console.log(shoppingCart.showItems());

⚠️ よくある間違いと正しい書き方

❌ よくある間違い

// プロパティ名に-を使う const obj = { user-name: "太郎" // エラー！ }; // thisを忘れる const person = { name: "太郎", greet: function() { return "Hello " + name; // undefinedになる } };

✅ 正しい書き方

// キャメルケースまたはクォートを使う const obj = { userName: "太郎", // OK "user-name": "太郎" // OK }; // thisを正しく使う const person = { name: "太郎", greet: function() { return "Hello " + this.name; // 正しい！ } };
