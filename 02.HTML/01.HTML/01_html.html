ウェブページの基礎を理解しよう

📖 HTMLとは？

HTML（HyperText Markup Language）は、ウェブページの構造と内容を定義するマークアップ言語です。

重要なポイント：
  • ウェブページの「骨組み」を作る言語
  • テキスト、画像、リンクなどの要素を配置
  • ブラウザが解釈して表示
  • プログラミング言語ではなく「マークアップ言語」

🏗️ HTMLの基本構造

HTMLドキュメントは決まった構造を持っています：

<!DOCTYPE html> <html lang="ja"> <head> <meta charset="UTF-8"> <title>ページタイトル</title> </head> <body> <h1>見出し</h1> <p>段落の内容</p> </body> </html>
html ├── head（ページの設定情報） │ ├── meta（文字エンコーディング） │ └── title（ページタイトル） └── body（ページの内容） ├── h1（見出し） └── p（段落）

🏷️ HTMLタグの仕組み

HTMLは「タグ」という仕組みで要素を定義します：

タグの基本形式

<タグ名>内容</タグ名>

属性付きタグ

<タグ名 属性="値">内容</タグ名>

タグの構成要素

  • 開始タグ： <タグ名>
  • 内容： タグに囲まれた部分
  • 終了タグ： </タグ名>
  • 属性： タグの追加情報

📝 主要なHTMLタグ

構造に関するタグ

<h1>〜<h6> : 見出し（レベル1〜6） <p> : 段落 <div> : 汎用ブロック要素 <span> : 汎用インライン要素 <header> : ヘッダー領域 <main> : メインコンテンツ <footer> : フッター領域

リンクと画像

<a href="URL">リンクテキスト</a> : リンク <img src="画像のURL" alt="代替テキスト"> : 画像

リストとテーブル

<ul> <ol> <table> <li>項目</li> <li>項目</li> <tr> </ul> </ol> <td>データ</td> </tr> </table>

🌟 HTMLとCSS、JavaScriptの関係

ウェブページの3つの要素：

  • HTML： 構造と内容（骨組み）
  • CSS： デザインとスタイル（見た目）
  • JavaScript： 動的な機能（動作）
<!-- HTML: 構造 --> <p class="highlight">重要な情報</p> <!-- CSS: スタイル --> <style> .highlight { color: red; font-weight: bold; } </style> <!-- JavaScript: 動作 --> <script> document.querySelector('.highlight').onclick = function() { alert('クリックされました！'); }; </script>

✅ HTMLを学ぶためのステップ

学習の進め方：

  1. 基本タグを覚える： h1, p, div, a, img など
  2. 構造を理解する： head, body の役割
  3. 属性を学ぶ： class, id, href, src など
  4. 実際に書いてみる： 簡単なページを作成
  5. CSSと組み合わせる： デザインを追加
  6. JavaScriptで機能追加： インタラクティブに

🔧 実践的なサンプルコード

簡単な自己紹介ページの例：

<!DOCTYPE html> <html lang="ja"> <head> <meta charset="UTF-8"> <title>私の自己紹介</title> </head> <body> <header> <h1>田中太郎のページ</h1> </header> <main> <section> <h2>自己紹介</h2> <p>こんにちは！プログラミングを勉強中です。</p> <h3>趣味</h3> <ul> <li>プログラミング</li> <li>読書</li> <li>映画鑑賞</li> </ul> </section> <section> <h2>連絡先</h2> <p>メール: <a href="mailto:tanaka@example.com">tanaka@example.com</a></p> </section> </main> <footer> <p>© 2024 田中太郎</p> </footer> </body> </html>
